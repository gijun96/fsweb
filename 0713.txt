 URL은 인터넷에서 웹 페이지,이미지, 비디오 등 리소스의 위치를 가리키는 문자열

프로토콜은 컴퓨터 내부에서, 또는 컴퓨터 사이에서 데이터의 교환 방식을 정의하는 규칙체계입니다.

원페이지 웹사이트
웹 사이트: 페이지 모음
홈페이지
메인페이지
서브 페이지: 로그인, 회원가입, 기업정보, 브랜드 소개등

화면 확대 축소: ctrl + '+'/'-'

<시험문제> 비주얼 스튜디오 코드의 설치 및 실행 정차를 서술하시오.
1단계: 비주얼 스튜디오 코드 프로그램을 다운받아 설치한다.
2단계: 확장 프로그램을 설치한다.
3단계: 사용자 환경에 맞게 옵션을 설정한다.
      테마, mouse wheel, wordwrap, 라이브서버 단축키 변경
4단계: 작업 영역을 설정한다.
5단계: 코드를 작성하고 웹 브라우저로 실행한다.


1.비주얼 스튜디오 코드(html + css + Javascript)
2. 추가 설치 -> 확장 프로그램(Plug in) 

파일: index.html, style.css, script.js <- 파일명.확장자
